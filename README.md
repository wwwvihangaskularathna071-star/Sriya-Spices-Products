# Sriya Products Website

This is a complete, responsive website for Sriya Products with a built-in Admin Dashboard.

## Features
- **Public Website**: Home, Products, About Us, Contact.
- **Admin Dashboard**: Manage products (Add/Edit/Delete), update company information.
- **No Database Setup Required**: Uses your browser's local storage to save data.
- **Responsive Design**: Works on mobile and desktop.

## How to Run
1. Simply double-click `index.html` to open it in your web browser.
2. Navigate through the site.

## Admin Access
- Click "Admin Login" in the menu (or footer on mobile).
- **Username**: `admin`
- **Password**: `admin123`

## Managing Images
To add your own product images:
1. Copy your image files into the `images` folder.
2. In the Admin Dashboard when adding a product, enter the path like: `images/my-spice.jpg`.
3. Alternatively, you can use image URLs from the internet.

## Technologies Used
- HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- FontAwesome for icons
- Google Fonts (Playfair Display, Poppins)
